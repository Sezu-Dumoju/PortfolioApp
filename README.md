# Portfolio App — Bohdan

Aplikacja mobilna portfolio studenta zbudowana w React Native / Expo.

## Funkcjonalności
- Ekran profilu z danymi, opisem i umiejętnościami
- Lista projektów zapisywana lokalnie (AsyncStorage)
- Szczegóły projektu z linkiem do GitHub
- Formularz dodawania projektu z walidacją danych
- Ekran kontaktowy z linkami
- Nawigacja zakładkowa i stosowa z ikonami

## Instrukcja uruchomienia
1. Zainstaluj Node.js: https://nodejs.org
2. Zainstaluj Expo Go na telefonie (App Store / Google Play)
3. Sklonuj repozytorium:
   git clone https://github.com/Sezu-Dumoju/PortfolioApp.git
4. Wejdź do folderu i zainstaluj zależności:
   cd PortfolioApp
   npm install --legacy-peer-deps
5. Uruchom aplikację:
   npx expo start
6. Zeskanuj QR kod w aplikacji Expo Go

## Technologie
- React Native
- Expo SDK 54
- React Navigation (Bottom Tabs + Stack)
- AsyncStorage
- Expo Vector Icons
